# Applied Improvisation Definition Generator

A simple interactive web app based on the **Applied Improvisation Definition Generator** created by **Ted DesMaisons** with **[Rebecca Stockley](https://www.improvlady.com/)**, November 2014, published by **[ANIMA · Inspired Learning](https://animalearning.com/)**.

## What it does

Pick one word or phrase from each of the six columns to build your own custom definition of Applied Improvisation:

> **[ing verb]** the **[noun]** ...of improvisational theatre to... **[verb]** **[object]** **[location]**

For example:
> *Harnessing the wisdom ...of improvisational theatre to... unleash transformation in YOU!*

Use the **Randomize** button to generate a surprise definition, or hand-pick each word yourself. Hit **Copy Definition** to grab your sentence and share it.
